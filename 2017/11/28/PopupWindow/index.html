PopupWindow 的方法

发表于 2017-11-28 | 分类于 Android

PopupWindow 的四个常用方法

1、setTouchable(boolean touchable)

设置 PopupWindow 是否可以响应触摸或者点击事件（默认参数为 true），参数 true 所有项可以响应，否则为 false 无法响应。

2、setFocusable(boolean focusable)

设置 PopupWindow 是否有获取焦点的能力（默认参数为 false），但是普通控件不需要获取焦点，这个方法只是针对于 EditText 的，EditText 若是没有焦点则是无法让我们编辑的，若 PopupWindow 有 EditText 的子项，调用这个方法将参数设置为 true OK。

3、setOutsideTouchable(boolean touchable)

顾名思义，这个方法是设置当 PopupWindow 处于活动状态时，PopupWindow 以外的区域是否可点击，参数为 true 的情况下点击 PopupWindow 以外的区域，PopupWindow 就会消失，参数为 false 的时候点击 PopupWindow 以外的区域是没有任何反应的。
注意：
设置这个方法的时候，需要在这个方法之前调用 setBackgroundDrawable(Drawable background) 方法，传入一个空的 bitmap，否则当你设置 setOutsideTouchable(boolean touchable) 参数为 true 时，点击外部区域一样是未有任何反应的。

4、setBackgroundDrawable(Drawable background)

这个方法一看就可以知道这是个设置背景的返回合法，在方法 3 得知，加上这个函数才可以使 setOutsideTouchable() 生效，并且，只有加上它之后，PopupWindow 才会对手机的返回按钮有响应，不加的话，点击手机返回按钮会退出当前 Activity 而不是退出 PopupWindow，此外，对于 PopupWindow 上层没有捕捉的点击事件，点击之后，仍然能使窗体消失。
