BaseAdapterHelper 介绍

BaseAdapterHelper 的 Github 地址
每当遇到一个 ListView 的时候，都要为它写一个自定义的 Adapter，这样可能一个项目要写很多的 Apapter，Apapter 的内容有很大一部分相似，就是处理数据和 getView，然后呢 BaseAdapterHelper 是 Adapter 的框架，只需要创建 QuickAdapter，传入布局和数据，绑定控件和显示的数据即可。
官方给的实现内容：

在 GitHub 官网下载下来的项目有四个文件：
BaseAdapterHelper 可以使得 BaseAdapter 的 getView() 方法更加简单的使，摆脱ViewHolder模板的写法
BaseQuickAdapter QuickAdapter 的高层类封装，实现BaseAdapter中抽象方法，同时对于数据的操作以及其他操作
QuickAdapter 该类可以让我们自定义的 BaseAdapter 类的代码大量的缩减，让我们把精力集中在业务实现上面，我们只需要做的是绑定控件 view 和数据即可
EnhancedQuickAdapter 增强的 QuickAdapter，主要是用于绑定控件 view 和数据
